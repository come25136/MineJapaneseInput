MineJapaneseInput
--

### MineJapaneseInput 通称 MJI とは
Minecraft1.9 からサポートされた日本語入力を補助するアプリケーションです。

### 使い方
"t"キーを押すことで"全角モード"へ切り替わり、"Enter"キーを2回続けて押すことにより"半角モード"へ切り替わります。
"/"キーを押した場合は切り替わらないようになっています。
